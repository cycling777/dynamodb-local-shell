# dynamodb-local-shell
You can try dynamodb-local with console instantly.

# Requirements
- python3
- git
- docker
- docker-compose

# Usage
### On your local PC
```
git clone https://github.com/cycling777/dynamodb-local-shell.git
cd dynamodb-local-shell
docker-compose up -d
```
### On browser
```
localhost:8000/shell
```

# Stop Runnning
### on your local machine
```
docker-compose down
```
